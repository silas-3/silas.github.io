Código fonte / Pré-formatação ( liguagem mono-epaçada <code></code> )

O comando document.getElementById (teste) é escrito em linguagem JavaScript; e utilizamos a tag <code></code>.

utilizamos a tag <code></code> para deixar todas as letras e caracteres terem o mesmo tamanho no quesito de ocupar espaço, ajuda tambem na leitura das letras mais estreita, deixando o texto mono-espaçado. 

                                                                                DICA
    se utilizamos a tag <pre></pre>; podemos centralizar o paragrafo como quisermos e podemos quebrar a linha sem utilizar a tag <br>lt;/br> varias vezes, deixando o codigo mais limpo e
     organidazado e de facil leitura.
               OBS.: <pre></pre> com <code></code>, é correto usar com a liguagem JavaScript; já que a tag <code></code> é escrita em linguagem JavaScript.

Citações (função <q></q> )

a função <q></q> cria ( ) em torno da fraze, daria para colocar direto as (" "), porém ficaria de maneira incorreta, pois sem a tag ela nao teria sentido, porque com a tag ela da sentido ao caractere.

OBS.: 

só usamos para Citações a tag  <q></q>.

Abreviações (<abbr title=" "></abbr>)

quando passamos o cursor do maue por cima da palavra abreviada, ela gera o nome em um balão.

Exemplo:

a sigla de html e css, são Abreviações. porém usando a tag <abbr></abbr> poderemos ler o significado da abreviação: html - css
