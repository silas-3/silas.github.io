Texto ( Grande <big></big> ) ( Pequeno <small></small> )

para deixar a letra do texto menor da para usar a tag: <small></small>.
para deixar grande da para usar por enquanto a tag: <big></big> porém ela (não é semântica) e já caiu em desuso e logo ira para de funcinar.

OBS.: dá para configurar o tamanho da letra utilizando o (.css), criando um selor para o texto.

OBS.: coloquei em negrito para ver melhor a mundança.

Texto deletado ( <del></del>)

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

para isso usamos a funçãos: <del></del>.

Texto inserido ( <ins></ins> )

Podemos marcar um texo como inserido uma ênfase e indicar que ele foi adicionado depois.

utilizando a tag: <ins></ins>

dá para utilizar por enquanto a tag <u> porém ela é (não se mantica) e logo pode cair em desuso.

Texto sobre-escrito( X2+5 <sup></sup> )

para inserir coisas do tipo: X2+5.

utiliza a tag: <sup></sup>.

exemplo:

x<sup>2</sup>+5

Texto sob-escrito ( H2O <sub></sub> )

para inserir coisas do tipo: H2O.

utiliza a tag: <sub></sub>
