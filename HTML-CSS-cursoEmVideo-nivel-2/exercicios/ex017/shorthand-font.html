Font Derivações.

font-weight: bold

quando criamos um (font-weight:) podemos definilo por numero ou por nome, sendo eses nomes:

  • ligther..
  • normal.
  • bold.
  • bolder.

    • font-style:italic.

    Quando criamos a função (font-style) podemos inseri o comando italic tracuzindo: (italico).

    text-decoration: underline.

    OBS.: para poder inserir essas configurações, temos que ver se a familha da fonte permite essas configuração.

    Shorthand-font. (forma abreviada)

    Podemos fazer todas essas variações com  uma Shorthand-fon, porém temos que seguir uma hieraquia no inserimento dos nomes.
      sendo font: (style) (weight) (size) (family)
      ex:
        
    
    1.  font: italic bolder 3em 'work Sans' 
                                
    
                    ou
    OBS.: podemos tirar algumas funções, porém a hierarquia tem que ser seguida. 
            
      ex:
        
    1. font: bolder 3em 'work Sans'
    OBS.: não existe shorthand pra tudo, porém existe varias que podem ser utilizados para HTML.
