Trabalhando com fontes

Foi utilizado esté estilo de css local, para mostrar os módelos de font e suas particularidades. 

            <style>
                body {
                    font-family: 'Courier New', Courier, monospace;
                }
                h1,h2 {
                    font-family: 'Times New Roman', Times, serif;
                }
            </style>

Neste exémplo, são adicinadas, três tipos de catecorias de letras, este é um modelo de fonte segura, pois alguns modelos de aparelhos não possui certos tipos de fontes, ai é adicionado mais dois modelos, para que caso o primeiro falhe, ele entenda o segundo, se não der, ele vai pro terceiro, ai ele da certo.

Como sabemos quais os modelos seguros; é só acessar o site da 3wschool, pesquisar os modelos de fontes.
