Centralinzando com a função(text-align)


Podemos posicionar as tag aonde quisermos, basta colocarmos a função no seletor da css { text-align= ****; }

Podemos também dar um espaço identificado usando com uma medida, com exemplo em um começo de parágrafo com a função no seletor { text-indent: 30px; } fazendo com que crie uma distância no começo do parágrafo, de acordo com a medida que inserirmos.

EX de como ficou o resultado:

        
            body {
                font: normal 16px Arial, Verdana, serif;
                text-align: left;
            }
            h1 {
                text-align: center;
            }
            h2 {
                text-align: right;
            }
            p {
                text-align: justify;
                text-indent: 30px;
            }

OBS.: nunca alinhar usando HTML, sempre utilizar CSS.
