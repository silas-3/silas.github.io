explicando como posicionar mais de uma tag repitida em locais diferentes

Iremos criar um index em CSS para fazer essas mudanças de posição nos titulo h1, h2 e parágrafos.

Iremos criar uma identificação para as tags em HTML, por ex: <h1 id= ****>

Identificação para CSS é utilizado #----- 

 exemplo de como ficou o nosso index em css:

            @charset "UTF-8";

body {
    background-color: rgba(193, 255, 101, 0.853);
}

h1 {
    color: rgb(127, 188, 37);
}

h1#principal { text-align: center;

    color: palevioletred;

    background-color: rgb(254, 206, 222);

}

h2 {
    color: rgba(77, 129, 0, 0.87);
}

Em html o comando para dar nome a tag é id= "" e em CSS é #nomedoid

OBS.: o certo é ter apenas um id por tag, se quiser a mesma configuração para mais de uma tag, o comando correto e class, e para usar ele em CSS, tem que comer com ponto: .nomedoseletor
