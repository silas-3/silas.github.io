TEXTO ESCONDIDO/INVISIVEL.

Utilizamos essa configuração para esconder o texto: 

            div > p {
                display: none;
                
            }
            
            div:hover > p {
                display:block;

color: rgb(7, 0, 219);

background-color: yellow;
                    
                    
            }
            
            div:hover {
                color: rgb(219, 0, 219)
            }

No exemplo superior utilizamos uma <div></div> e inserimos um <p></p> dentro dela. ai fizemos uma configuração usando a função > na css para dizer que tem um parágrafo dentro da div. ai fizemos essa configuração na css.

OBS.: >
Quando inserimos > na css queromos dizer como se fosse um filho dentro de algo.
