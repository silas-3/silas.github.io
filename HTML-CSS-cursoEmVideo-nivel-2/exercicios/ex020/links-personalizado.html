            a:visited
Ao visitar a página pela primeira vez, ela muda a cor para indicar que já abrimos o link alguma vez.
{
                color: rgb(99, 4, 99);
            }
        
            a:hover
Deixando estilizado para ter interação ao passar o cursor sobre o link, ele mudar para o estilo configurado.
{
                color: yellow;
                text-decoration: underline;
            }
        
            a:active
Ao ativarmos o link ele muda de cor rapidamente para mostrar que clicamos nele, se você clicar e segurar, para não abrir, ira ver a diferença de cor.
{
                color: rgb(91, 0, 0);
            }
        
            .especial
Esta foi a criação de uma class epecifica que eu quis adicionar uma configuração apenas pra ela, ai nomeamos ela como: .especial
{
                background-color: rgb(0, 255, 0)
            }
        
            .especial::before{
                content: '⇒';
            }
        
            .especial::after{
                content:'⇐' ;
