Fontes Externa/baixada

@font-face ex:

        @font-face {
            font-family: ;
            src: url(); ai podemos inserir mais estilo: format()
            font-weight: normal;
            font-style: normal;
        }
    

            Tipos de format()
            - opentype (otf)
            - truetype (ttf)
            - embedded-opentype
            - truetype-aat (Apple Advanced Typography)
            - svg
