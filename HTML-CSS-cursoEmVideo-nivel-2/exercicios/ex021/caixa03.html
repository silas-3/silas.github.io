Bordas personalizas

Geralmente fazemos ela por um editor e inserimos a imagem

COIDIGO INSERIDO 

h1{
            border: 15px solid black;
            padding: 10px;
            border-image-source: url('borda-personalizada-gimp.png');
            border-image-slice: 150;
            border-image-repeat: repeat;
            border-width: 50px;
            
            ou podemos fazer uma shorthand.

            border-image: url('borda-personalizada-gimp.png') 150 repeat;
            
        }
