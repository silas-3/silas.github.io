Explicação

Criamos a tag <style></style> abaixo da tag <title></title> 

        <style>
    
        *{ [ *{} ] é a configuração globais das css
            height: 100%;
        }
    
        body { background-image: linear-gradient(to bottom, #4F5FD1,#B030A6, #CE2F77,#EC5E41,#FB9734,#FED97D);
                background-attachment: fixed;
    
            }
    
        </style>

*{} é a configuração globais das css

O comando background-image, permite com que insera a cor de fundo da page, adicionando o comando background-image: linear-gradient, ai é possivel mesclar quantas cores quiser, tanto da esquerda para a direita, quanto de cemia para baixo e ou vise e versa, so é preciso inser esse comando dentro () e em inglés, exemplo background-image: linear-gradient(to bottom, white, blue);

porém ao utilizar, será necesario adicinar mai um comando para deixar a página fixa, ou deixar a página compactada, endo esse comando background-attachment: fixed;. Esté comando fixa a pagina, quando deslizar a pagina o fundo fica fixo.

Irei colocar o fundo do instagram como background-color na tag <body> porém para fazer esse degrade irei usar o comando background-image: linear-gradient:

  1. #4F5FD1
  2. #B030A6
  3. #CE2F77
  4. #EC5E41
  5. #FB9734
  6. #FED97D

OBS.:sempre quando for utilizar paletas, utilizar no máximo de 3 a 5 cores, para não ficar muita cor e para não faltar cor.
